# Vault North 1.0

Welcome!

Vault North is a bank management software that handles:

* Customers
  * Account numbers
  * Debit cards

* Transactions

### The functionality includes:

* Search
  * Customer
* Customer overview
 * Transaction history
 * Accounts
 * Cards
 * Loans

### Server side:
* ASP .NET Core 3.1
* SQL Server
* Entity Framework
* MS Test
  * Moq

### Client side:
* HTML
* CSS
  * Bootstrap
* jQuery
